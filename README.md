# Zakat Gold Calculator (ICT602 Lab)

A mobile application developed for the ICT602 course to calculate gold zakat based on weight and current market price.

## Features
- Calculate total gold value.
- Calculate Zakat payable value based on Uruf (Keep/Wear).
- Calculate total Zakat (2.5%).
- Input validation and error handling.
- About page with developer info.
- Sharing functionality.
